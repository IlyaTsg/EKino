Про функционал:
	При запуске программы данные сразу будут считываться с файла
	Потом их можно будет дополнять\удалять 

	Два фильма с одним названием нельзя добавить

	Поиск фильма осуществляется по всем параметрам

	Сортировка по дате, рейтингу и кол-ву звёзд

	Вывод в формате таблицы

	|Title|Director|Date|etc..|
	---------------------------
	|Titanic|J. Cameron|1.1.1|...|
	---------------------------
	|Dead Poet Cosity|Ilya Ts|1.2.3|...|
	--------------------------- 

Структура программы:
	Для хранения фильмов используется двусвязный линейный список

	Функция FileEnter создаёт и заполняет список, возвращает указатель на первый эелмент данного списка
